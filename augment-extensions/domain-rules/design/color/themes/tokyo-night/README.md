# Tokyo Night Theme

Dark theme inspired by Tokyo's neon-lit nights.

## Colors

- **Primary**: `#7aa2f7` (Blue)
- **Secondary**: `#bb9af7` (Purple)
- **Background**: `#1a1b26` (Dark Navy)
- **Surface**: `#24283b` (Navy)
- **Text**: `#c0caf5` (Light Blue)
- **Accent**: `#7dcfff` (Cyan)
- **Success**: `#9ece6a` (Green)
- **Warning**: `#e0af68` (Orange)
- **Error**: `#f7768e` (Pink)
- **Info**: `#7aa2f7` (Blue)

## Usage

```bash
augx link themes/tokyo-night
```
