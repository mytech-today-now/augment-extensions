# Screenplay Quick Reference Guide

## Fountain Syntax Cheat Sheet

### Scene Headings
```fountain
INT. LOCATION - DAY
EXT. LOCATION - NIGHT
INT./EXT. LOCATION - DUSK
```

### Character & Dialogue
```fountain
CHARACTER
Dialogue goes here.

CHARACTER (O.S.)
Off-screen dialogue.

CHARACTER (V.O.)
Voice-over narration.

CHARACTER
(parenthetical)
Dialogue with direction.
```

### Action
```fountain
Character does something.

Sound effects in ALL CAPS.

Keep paragraphs SHORT.
```

### Transitions
```fountain
CUT TO:

FADE OUT.

DISSOLVE TO:
```

## Three-Act Structure

### Act 1 (Setup) - Pages 1-25
- Introduce protagonist
- Establish world
- Inciting incident (p.10-12)
- Plot Point 1 (p.25-30)

### Act 2 (Confrontation) - Pages 26-85
- Rising action
- Midpoint (p.55-60)
- Complications
- Plot Point 2 (p.75-85)

### Act 3 (Resolution) - Pages 86-110
- Climax
- Resolution
- Denouement

## Dialogue Best Practices

**DO:**
- Keep it natural
- Use subtext
- Make each character distinct
- Read aloud to test
- Cut unnecessary words

**DON'T:**
- Info-dump
- Use on-the-nose dialogue
- Make everyone sound the same
- Over-explain
- Use dialogue for exposition

---

## Action Writing Tips

**DO:**
- Be visual and specific
- Keep paragraphs short (3-4 lines)
- Use active voice
- Show, don't tell
- Write what we SEE and HEAR

**DON'T:**
- Write camera directions
- Over-describe
- Tell us what characters think/feel
- Use passive voice
- Write novelistic prose

---

## Common Mistakes to Avoid

1. **Too much description** - Keep it visual and brief
2. **Unnatural dialogue** - Read it aloud
3. **Camera directions** - Write what we see, not how
4. **Telling emotions** - Show through action
5. **Inconsistent formatting** - Follow standards
6. **Too long** - Respect page count guidelines
7. **Weak structure** - Use beat sheet
8. **Flat characters** - Develop fully

## Quick Tips

1. **Write visually** - What do we SEE?
2. **Keep it moving** - Cut anything that slows pace
3. **Show, don't tell** - Action reveals character
4. **Read aloud** - Dialogue must sound natural
5. **Follow format** - Industry standards matter
6. **Know your category** - Different rules apply
7. **Revise ruthlessly** - First draft is just beginning
8. **Get feedback** - Fresh eyes catch issues

---

## Emergency Fixes

**Dialogue feels flat?**
→ Add subtext, cut exposition, make it specific

**Pacing drags?**
→ Cut description, shorten scenes, raise stakes

**Characters sound same?**
→ Define distinct voices, add quirks, vary patterns

**Structure unclear?**
→ Use beat sheet, identify turning points, clarify acts

**Too long?**
→ Cut subplots, tighten action, eliminate redundancy

**Formatting wrong?**
→ Use Better Fountain, check examples, validate against standards
