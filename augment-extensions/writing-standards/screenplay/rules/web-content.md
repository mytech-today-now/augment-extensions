# Web Content

## Overview

Web content for platforms like YouTube, TikTok, Instagram, and other digital platforms requires immediate hooks, platform-specific optimization, and engagement-driven storytelling designed for short attention spans and algorithm-driven distribution.

## Platform-Specific Formats

### YouTube
**Short-Form** (1-5 minutes):
- Quick tutorials
- Comedy sketches
- Reaction videos
- Vlogs

**Mid-Form** (5-15 minutes):
- Detailed content
- Narrative stories
- Documentary-style
- Educational content

**Long-Form** (15+ minutes):
- In-depth analysis
- Extended narratives
- Interview format
- Series content

### TikTok/Instagram Reels/YouTube Shorts
**Ultra-Short** (15-60 seconds):
- Immediate hook (first 3 seconds)
- Single concept/joke
- Vertical format (9:16)
- Text overlays
- Trending audio

### Instagram/Facebook
**Stories** (15 seconds per slide):
- Episodic content
- Behind-the-scenes
- Quick updates
- Interactive elements

**Feed Videos** (1-3 minutes):
- Polished content
- Square or vertical format
- Captions required
- Thumbnail optimization

## Core Principles

### The 3-Second Rule
**Hook Immediately**:
- Visual impact in first frame
- Intriguing question
- Shocking statement
- Action in progress
- Emotional trigger

**Bad Opening**:
```
FADE IN:

INT. BEDROOM - DAY

Sarah wakes up, stretches, and gets out of bed.
```

**Good Opening**:
```
SMASH CUT TO:

Sarah's phone EXPLODES with notifications. Her face
goes pale as she reads the screen.
```

### Engagement Optimization
**Techniques**:
- Pattern interrupts (every 5-10 seconds)
- Visual variety
- Text overlays for key points
- Music/sound effects
- Emotional peaks and valleys
- Call-to-action

### Retention Strategies
**Keep Viewers Watching**:
- Tease payoff early
- Build curiosity loops
- Use cliffhangers
- Promise value
- Deliver quickly
- Avoid slow burns

## Structure for Short-Form

### 15-60 Second Format
**Structure**:
1. **Hook** (0-3 seconds): Grab attention
2. **Setup** (3-10 seconds): Establish premise
3. **Development** (10-45 seconds): Build/escalate
4. **Payoff** (45-60 seconds): Deliver resolution
5. **CTA** (final 3 seconds): Like/follow/comment

**Example**:
```
HOOK (0-3s):
"This simple trick will change your life."

SETUP (3-10s):
Show problem everyone relates to.

DEVELOPMENT (10-45s):
Demonstrate solution step-by-step.

PAYOFF (45-60s):
Show amazing result.

CTA (final 3s):
"Follow for more tips!"
```

### 1-5 Minute Format
**Structure**:
1. **Hook** (0-10 seconds)
2. **Promise** (10-20 seconds): What viewer will get
3. **Content** (20 seconds - 4 minutes): Deliver value
4. **Recap** (final 30 seconds): Summarize key points
5. **CTA** (final 10 seconds): Subscribe/like/comment

## Viral Hooks

### Types of Hooks
**Question Hook**:
- "What would you do if..."
- "Have you ever wondered..."
- "Why does..."

**Shock Hook**:
- Unexpected visual
- Surprising statement
- Dramatic moment
- Controversial claim

**Relatability Hook**:
- "POV: You're..."
- "When you..."
- "That feeling when..."

**Curiosity Hook**:
- "You won't believe..."
- "The secret to..."
- "What happens next..."

**Value Hook**:
- "3 ways to..."
- "How to..."
- "The best..."

## Visual Storytelling

### Vertical Format (9:16)
**Composition**:
- Center important elements
- Use full vertical space
- Text in safe zones
- Face close-ups
- Dynamic framing

**Avoid**:
- Horizontal compositions
- Small text
- Wide shots
- Complex visuals

### Text Overlays
**Best Practices**:
- Large, readable fonts
- High contrast
- Animated text
- Key words emphasized
- Captions for accessibility
- Emoji for emphasis

**Placement**:
- Top third: Headlines
- Middle: Main content
- Bottom third: Captions/CTA

### Visual Pacing
**Techniques**:
- Quick cuts (every 2-3 seconds)
- Zoom transitions
- Jump cuts
- B-roll inserts
- Graphics and animations
- Color grading shifts

## Audio Considerations

### Music Selection
**Purpose**:
- Set tone and mood
- Increase energy
- Trending audio (TikTok/Reels)
- Copyright-free options
- Match pacing

### Sound Effects
**Usage**:
- Emphasize moments
- Create comedy
- Build tension
- Transition scenes
- Enhance impact

### Voiceover
**Delivery**:
- Energetic and engaging
- Clear enunciation
- Conversational tone
- Paced for retention
- Emotional variation

## Platform Algorithms

### YouTube Algorithm
**Optimization**:
- Click-through rate (thumbnail + title)
- Watch time (retention)
- Engagement (likes, comments, shares)
- Session time (keeps viewers on platform)
- Consistency (regular uploads)

### TikTok/Reels Algorithm
**Optimization**:
- Completion rate (watch to end)
- Re-watches
- Shares
- Comments
- Trending audio/hashtags
- Post timing

### Engagement Metrics
**Track**:
- View duration
- Drop-off points
- Engagement rate
- Share rate
- Click-through rate
- Conversion rate

- "Follow for more"
