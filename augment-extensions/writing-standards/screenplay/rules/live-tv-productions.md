# Live TV Productions

## Overview

Live television productions require real-time scripting, multi-camera coordination, precise timing, and contingency planning for unpredictable elements. Scripts must balance structure with flexibility for improvisation and live audience interaction.

## Types of Live Productions

### Live Variety Shows
- Saturday Night Live
- Late-night talk shows
- Award shows
- Sketch comedy
- Musical performances

### Live Sitcoms
- Multi-camera setup
- Live studio audience
- Theatrical staging
- Continuous performance
- Immediate feedback

### Live Events
- Sports broadcasts
- News coverage
- Award ceremonies
- Concerts
- Special events

### Live Drama
- Live theater broadcasts
- One-night television events
- Experimental productions
- Holiday specials

## Script Format

### Multi-Camera Script Format
**Format**:
```
FADE IN:

INT. LIVING ROOM - NIGHT

                    JOHN ENTERS FROM KITCHEN, CARRYING PIZZA BOX.

                         JOHN
          (TO AUDIENCE)
     Who ordered pepperoni?

                    AUDIENCE LAUGHS.

                         MARY
          (SARCASTIC)
     Oh great, more pizza.

                    AUDIENCE LAUGHS.

                         JOHN
          What? You love pizza!

                         MARY
          I love NOT having pizza every night!

                    AUDIENCE APPLAUSE.
```

**Characteristics**:
- ALL CAPS for scene headings
- Centered character names
- Centered dialogue
- Centered parentheticals
- Audience reactions noted
- Camera cues in margins

### Timing Marks
**Notation**:
```
[0:00] COLD OPEN
[2:30] OPENING CREDITS
[3:00] MONOLOGUE
[8:00] COMMERCIAL BREAK 1
[11:00] SKETCH 1
[16:00] COMMERCIAL BREAK 2
[TOTAL RUNTIME: 60:00]
```

### Camera Cues
**Format**:
```
CAM 1: WIDE SHOT - LIVING ROOM
CAM 2: CLOSE-UP - JOHN
CAM 3: TWO-SHOT - JOHN AND MARY
CAM 4: AUDIENCE REACTION
```

## Live Audience Considerations

### Audience Warm-Up
**Pre-Show** (10-15 minutes):
- Comedian or host
- Explain show format
- Practice applause/laughter
- Build energy
- Set expectations

### Audience Cues
**Script Notation**:
```
                    APPLAUSE SIGN

                    AUDIENCE APPLAUDS.

                    LAUGHTER SIGN

                    AUDIENCE LAUGHS.
```

### Playing to Audience
**Techniques**:
- Pause for laughter
- Acknowledge reactions
- Eye contact with audience
- Physical comedy
- Audience interaction

### Audience Reactions
**Timing**:
- Allow time for laughter (3-5 seconds)
- Pause for applause (5-10 seconds)
- Don't step on laughs
- Adjust pacing in real-time
- Read the room

## Timing and Pacing

### Precise Timing
**Requirements**:
- Hit exact runtime
- Account for audience reactions
- Buffer time for delays
- Flexible segments
- Time cues for performers

### Timing Buffer
**Strategies**:
- Flexible segments that can be cut
- "Stretch" material if running short
- Pre-planned cuts if running long
- Real-time adjustments
- Stage manager time cues

### Time Cues
**Notation**:
```
STAGE MANAGER GIVES 5-MINUTE WARNING

STAGE MANAGER GIVES 1-MINUTE WARNING

STAGE MANAGER GIVES 30-SECOND WARNING

STAGE MANAGER GIVES WRAP CUE
```

## Improvisation and Flexibility

### Scripted vs. Improvised
**Balance**:
- Scripted structure
- Room for improvisation
- Ad-lib opportunities
- Spontaneous moments
- Controlled flexibility

### Improvisation Markers
**Script Notation**:
```
JOHN AND MARY AD-LIB ABOUT CURRENT EVENTS
(APPROXIMATELY 2 MINUTES)

[IMPROVISED BANTER]

JOHN RETURNS TO SCRIPT:
```

### Contingency Planning
**Prepare For**:
- Technical difficulties
- Performer mistakes
- Timing issues
- Unexpected events
- Backup material

## Multi-Camera Setup

### Camera Blocking
**Planning**:
- Pre-plan camera positions
- Choreograph camera moves
- Avoid crossing cameras
- Clear sightlines
- Smooth transitions

### Shot List
**Format**:
```
SCENE 1 - LIVING ROOM

SHOT 1: WIDE - Establish room
SHOT 2: MED - John enters
SHOT 3: CU - John's reaction
SHOT 4: TWO-SHOT - John and Mary
SHOT 5: CU - Mary's response
SHOT 6: WIDE - Both characters
```

### Camera Transitions
**Techniques**:
- Cut on movement
- Match action
- Reaction shots
- Audience cutaways
- Smooth flow

## Cue Cards and Teleprompter

### Cue Card Usage
**Guidelines**:
- Large, readable text
- Key lines only
- Positioned near camera
- Updated in real-time
- Backup for teleprompter

### Teleprompter
**Format**:
```
WELCOME TO THE SHOW!

[PAUSE FOR APPLAUSE]

TONIGHT WE HAVE A GREAT SHOW FOR YOU.

[GESTURE TO GUESTS]

PLEASE WELCOME OUR FIRST GUEST...
```

**Best Practices**:
- Short sentences
- Natural breaks
- Pause indicators
- Emphasis markers
- Easy-to-read font

## Error Recovery

### Handling Mistakes
**Strategies**:
- Acknowledge and move on
- Incorporate into comedy
- Have backup lines ready
- Stay in character
- Don't break unless necessary

### Technical Difficulties
**Contingencies**:
- Backup equipment
- Alternative segments
- Stall tactics
- Audience interaction
- Pre-recorded backup

### Script Notation
```
[IF TECHNICAL DIFFICULTY, GO TO BACKUP SKETCH]

[BACKUP MATERIAL - 3 MINUTES]

[RETURN TO MAIN SCRIPT WHEN READY]
```

## Live Show Structure

### Variety Show (60 minutes)
**Structure**:
- **Cold Open** (2-3 min): Sketch or monologue
- **Opening Credits** (1 min)
- **Monologue** (5-7 min): Host addresses audience
- **Commercial Break 1** (3 min)
- **Sketch 1** (5-7 min)
- **Commercial Break 2** (3 min)
- **Musical Performance** (4-5 min)
- **Commercial Break 3** (3 min)
- **Sketch 2** (5-7 min)
- **Commercial Break 4** (3 min)
- **Sketch 3** (5-7 min)
- **Goodnights** (2 min)

### Talk Show (60 minutes)
**Structure**:
- **Cold Open** (2 min)
- **Monologue** (8-10 min)
- **Commercial Break 1**
- **Guest 1** (10-12 min)
- **Commercial Break 2**
- **Guest 2** (8-10 min)
- **Commercial Break 3**
- **Musical Performance** (4-5 min)
- **Closing** (2 min)

## Best Practices

✅ Plan for contingencies
✅ Allow time for audience reactions
✅ Use clear timing marks
✅ Prepare backup material
✅ Communicate with crew
✅ Stay flexible
✅ Mark camera cues clearly
✅ Account for improvisation
✅ Build in timing buffers

### DON'T
❌ Over-script improvised moments
❌ Ignore timing constraints
❌ Forget audience reactions
❌ Neglect camera blocking
❌ Panic during mistakes
❌ Rush through material
❌ Forget backup plans
❌ Ignore stage manager cues
❌ Overcomplicate camera moves
❌ Sacrifice safety for comedy

## Script Example

**Live Sketch Comedy**:
```
SKETCH: "THE JOB INTERVIEW"
RUNTIME: 5:00
CAMERAS: 3

FADE IN:

INT. OFFICE - DAY

CAM 1: WIDE SHOT - OFFICE

                    BOSS SITS BEHIND DESK. APPLICANT ENTERS.

CAM 2: MED SHOT - APPLICANT

                         APPLICANT
          Thanks for seeing me.

CAM 3: CU - BOSS

                         BOSS
          Have a seat.

                    AUDIENCE LAUGHS (ANTICIPATORY).

CAM 2: TWO-SHOT

                    APPLICANT SITS. CHAIR COLLAPSES.

                    AUDIENCE LAUGHS.

CAM 3: CU - BOSS (DEADPAN)

                         BOSS
          That's a good start.

                    AUDIENCE ROARS.

[SKETCH CONTINUES...]

[4:30] STAGE MANAGER: 30-SECOND WARNING

[5:00] BLACKOUT

                    AUDIENCE APPLAUSE.
```
