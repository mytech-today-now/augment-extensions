# Changelog

All notable changes to this module will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-02-25

### Added
- **Modern Features**: Comprehensive guide to TypeScript 5.x features including decorators, const type parameters, and satisfies operator
- **Tooling Guide**: Modern tooling setup with ESLint flat config, Biome, tsup, and Vitest+MSW
- **Security & Performance**: Best practices for secure headers, type-safe runtime validation, memoization, and bundle optimization
- **Testing Strategies**: Comprehensive testing guide with unit/integration/e2e patterns and MSW for API mocking
- **Architecture Patterns**: Guide to folder organization, dependency inversion, and clean architecture
- **Monorepo Support**: Complete monorepo setup guide with Turborepo, pnpm workspaces, and changesets
- **Type Patterns**: Advanced type patterns including branded types, discriminated unions, and type-level programming
- **Examples**: Added eslint-flat-config.js, biome.json, and vitest-setup.ts examples

### Changed
- Expanded module from 6,416 to ~25,000 characters
- Updated description to reflect comprehensive coverage
- Reorganized file structure for better discoverability

### Improved
- Enhanced error handling patterns with modern approaches
- Updated naming conventions with latest best practices
- Improved cross-references between sections

## [1.0.0] - 2024-01-01

### Added
